# DirectToParticlePhoton
DirectToParticlePhoton is an open source software for Particle. It allows to connect Particle Photon wirelessly without necessity to use internet, from Android device, IOs Device or Microsoft mobile device. 

Its major functionality is to enable/disable all available pins (digital/analog) and allow to send/receive messages over RS232 protocol.
